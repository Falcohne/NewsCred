# NewsCred - Intelligent News Credibility Assessment Platform 
 
A mobile application that uses Natural Language Processing (NLP) and machine learning to analyze news articles and assess their credibility. 
 
## Features 
- Article credibility scoring (0-100 percent) 
- Fact-checking and claim verification 
- Date and author verification 
- Image verification 
- Premium subscription model 
- Dark mode support 
- User authentication 
 
## Tech Stack 
 
### Backend 
- Java 17 
- Spring Boot 2.7.18 
- PostgreSQL 
- JWT Authentication 
 
### Mobile 
- React Native (Expo) 
- TypeScript 
- React Navigation 
- React Native Paper
